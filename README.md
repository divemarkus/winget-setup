# winget-setup

PowerShell-based installer for winget and bulk app deployment.

## Usage

1. Open PowerShell **as Administrator**  
2. Clone this repo:  
   ```powershell
   git clone https://github.com/divemarkus/winget-setup.git
   cd winget-setup
3. Run the installer:
   ```powershell
   .\install-winget.ps1
